# Multi-Level Inheritance System (C++) 🚀👥

هذا المشروع يقدم نظاماً برمجياً متكاملاً لإدارة الموارد البشرية وهيكلية الموظفين، تم بناؤه لتطبيق المفاهيم المتقدمة في البرمجة كائنية التوجه (OOP) باستخدام لغة ++C.

## 📐 الهيكلية المعمارية للنظام (Architecture)
يعتمد النظام على وراثة متعددة المستويات لضمان إعادة استخدام الكود وتقليل التكرار:
`clsPerson` (الأب الأساسي) ⬅️ `clsEmployee` (الابن الأول) ⬅️ `clsProgrammer` (الحفيد)

## ✨ الميزات التقنية المطبقة:
- **Multi-level Inheritance**: تمرير البيانات وبناء الكائنات عبر تسلسل الـ Constructors الموروثة بشكل نظيف.
- **Method Overriding**: إعادة صياغة دالة الطباعة `Print()` في كل مستوى لعرض البيانات المخصصة لكل فئة.
- **Advanced Properties**: استخدام ميزة الـ `__declspec(property)` الخاصة بـ Microsoft C++ لمحاكاة الـ Getters والـ Setters كلغات البرمجة الحديثة (C# / Java).
- **Communication Methods**: محاكاة لإرسال الرسائل القصيرة `SendSMS()` والبريد الإلكتروني `SendEmail()` مدمجة داخل كلاس الشخص.

## 🚀 كيفية استخدام الكود (Usage)
الكود يحتوي على كلاسات جاهزة للاستخدام الفوري. يمكنك إنشاء كائن لمبرمج جديد وتمرير كامل بياناته الهيكلية والمالية والتقنية بأسلوب مباشر عبر الـ Constructor.
